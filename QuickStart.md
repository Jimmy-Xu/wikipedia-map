# change config

change `api_endpoint` in `js/api.js` to `http://localhost:3000/`

# start api server
```
$ cd api
$ node index.js
start api server on 3000
```

# start web server
```
$ python -m SimpleHTTPServer
Serving HTTP on 0.0.0.0 port 8000 ...
```

# open url in webbrowser
```
http://127.0.0.1:8000
```
